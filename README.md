# Life Expectancy Prediction using Machine Learning

## Project Overview
This project predicts life expectancy using WHO health and socio-economic data.

## Algorithms Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- SVM
- XGBoost

## Steps Performed
- Exploratory Data Analysis (EDA)
- Missing Value Handling
- Skewness Treatment
- Outlier Detection
- Feature Scaling
- Model Training & Evaluation
- Overfitting Analysis
- Model Comparison

## Best Model
Random Forest Regressor achieved the best performance.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn

## Dataset
WHO Life Expectancy Dataset from Kaggle.
